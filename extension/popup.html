Embed LLMsTxt

You can expose your llms.txt files via (you can use any name but the proposal is to use llms.txt and/or llms-full.txt like robots.txt):

1. Link tag 
<link rel="alternate" type="text/markdown" href="https://example.com/path/to/llms-full.txt" title="LLM-friendly version">
2. Meta tag 
<meta name="llms-txt" content="https://example.com/path/to/llms-full.txt">
